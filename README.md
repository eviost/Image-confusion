空间填充曲线混淆器 (Hilbert Image Scrambler)

一个基于 Hilbert 空间填充曲线 的新一代图片混淆工具。它能在保护图片隐私的同时，最大程度保留像素的空间相关性，使得混淆后的图片依然能够被 JPG/WebP 等算法高效压缩，且文件大小不会显著增加。

✨ 核心特性

🔒 隐私优先：纯前端运行，所有处理均在浏览器本地通过 Web Worker 完成，图片绝不上传至任何服务器。

📉 高压缩比：不同于传统的噪声混淆，基于 Hilbert 曲线的算法保留了像素的局部邻域特征，使得混淆图可以直接保存为 JPG 格式而不产生巨大的文件体积。

🎨 现代化 UI：

高级磨砂质感：全站采用 Glassmorphism 设计风格。

多主题支持：内置“深色/浅色”模式及专属的“初音未来 (Miku)”配色主题。

响应式设计：完美适配桌面端与移动端操作。

⚡ 极速处理：利用多线程 (Web Worker) 技术，处理 4K 大图依然流畅不卡顿。

🛡️ 智能防呆：

防重复混淆：智能检测图片是否已被混淆，防止用户误操作导致无法还原。

信标验证：内置隐形像素信标，解密时自动校验密码正确性，拒绝显示乱码。

🛠️ 实用工具：内置随机强密码生成器、全屏高清预览、一键重置原图等贴心功能。

🚀 如何使用

1. 混淆 (加密) 图片

点击或拖拽上传一张图片 (支持 PNG, JPG, WebP)。

在“混淆密钥”栏输入一个密码，或者点击“随机”按钮生成一个强密码。

确保操作模式选择为 【混淆 (加密)】。

点击 “开始处理”。

处理完成后，点击 “下载结果” 保存图片。

2. 还原 (解密) 图片

上传之前混淆过的图片。

输入 完全相同 的密钥。

将操作模式切换为 【还原 (解密)】。

点击 “开始处理”，图片将瞬间恢复原貌。

注意：如果密钥错误，程序会弹出提示并阻止显示乱码结果（除非您勾选了“强制显示”）。

🧠 技术原理

本工具利用 Hilbert 曲线 将二维图像像素映射为一维序列：

降维：通过 Hilbert 算法遍历图片所有像素，生成一个保留空间局部性的一维像素数组。

位移：根据用户密钥生成的哈希值，对这个一维数组进行循环移位。

重构：将移位后的像素按 Hilbert 路径重新填回二维图像中。

由于 Hilbert 曲线的特性，原图中相邻的像素在混淆后依然倾向于在空间上保持接近（虽然位置已变）。这使得图像的高频信息（噪点）比完全随机打乱要少得多，从而对压缩算法更加友好。
